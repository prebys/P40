* The lab manual was maintained in Overleaf and has been downloaded to the 'Labs' directory
* The Canvas site has been exported to the 'Canvas Export' directory and split into smaller files for github
* The 'GradeAna Mods' directory contains a modified version of the GradeAna code and the final plots for grade analysis
